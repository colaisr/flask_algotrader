AlgoTrader Package

7. Ready! Launch with "win_start_all.bat"

Setting a PC

Adding to Autostart

1. Go to algotrader folder. Right-click win_start_all.bat, create shortcut. Cut/Copy it. 2. Go to C:\Users\Administrator\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup 3. Paste shortcut.

Setting Autorestart

1. Open Task Scheduler 2. Create Basic task. Action: shutdown Argument: /r

Setting AutoLogOn

1. Right click on Start and select Run, type NetPLWiz and then press Enter. 2. Select User Account. Uncheck the option to ask for a password.
